# About Ize Studio

Ize Studio is an English-language archive for essays on writing, reading, typewriters, e-ink tools, and the making of Ize Compose.

It gathers translated versions of selected Brunch posts alongside notes on books, writing instruments, project records, and the broader practice of building tools for thoughtful composition.

The site is not limited to Ize Compose. It is a home for connected work: essays, reading notes, translation drafts, writing systems, e-ink experiments, and project documentation.
