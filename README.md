# Ize Studio

Essays on writing, reading, typewriters, e-ink tools, and the making of Ize Compose.

This repository hosts the GitHub Pages site for Ize Studio, including English essays, translations from Brunch posts, reading notes, writing tool reflections, and project records.

## Categories

- Essays
- Writing Tools
- Reading Notes
- Projects
- Translations

## Writing Posts

Add Markdown files to `_posts` using the Jekyll filename format:

```text
YYYY-MM-DD-title.md
```

Each post should include front matter:

```yaml
---
layout: post
title: "Post Title"
date: 2026-06-03
category: Essays
---
```
